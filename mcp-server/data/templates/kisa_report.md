## 3. 유출등 피해 최소화를 위해 정보주체가 할 수 있는 방법 등

고객님께서는 다음 조치를 취해주시기 바랍니다:

1. **비밀번호 즉시 변경**
   - 당사 서비스 비밀번호 변경
   - 동일 비밀번호 사용 중인 다른 서비스도 변경

2. **2단계 인증 활성화**
   - 계정 설정에서 2단계 인증 설정 권장

3. **이상 거래 모니터링**
   - 본인 명의의 금융거래 내역 확인
   - 의심 거래 발견 시 즉시 금융기관에 신고

4. **피싱 주의**
   - 사고를 빌미로 한 피싱 이메일/문자 주의
   - 당사는 전화/문자로 개인정보를 요구하지 않습니다
